# Home Network Topology Lab

## Project Overview

I have created this home network topology lab to strengthen my networking fundamentals and improve my understanding of how traffic flows through a network from a SOC analyst's perspective.


## Objectives

- Build a basic home network topology.
- Understand the role of the ISP modem, firewall, and wireless router.
- Analyze network traffic from a client device to a web server.
- Demonstrate foundational networking knowledge for Security Operations Centre (SOC) analysis.


## Network Topology

The network consists of:

- ISP Modem
- Firewall
- Wireless Router (192.168.1.1)
- Laptop
- Smartphone
- Smart TV

## Network Traffic Analysis

Traffic Flow:

1. User requests a web page.
2. The wireless router forwards the packet.
3. The firewall inspects the traffic.
4. The ISP modem sends the traffic to the Internet.
5. The web server processes the request.
6. The response returns to the client device.

## Security Considerations

This lab highlights several basic security concepts:

- Firewalls help filter network traffic.
- IP addresses identity the device connected to the network to communicate.
- Routers perform Network Address Translation (NAT).
- Home devices should receive regular security updates to avoid threats incidents/attacks.
- Strong Wi-Fi security (WPA2/WPA3) should be enabled.
